# Microsoft Azure

O Microsoft Azure é uma poderosa plataforma de computação em nuvem da Microsoft, composta por uma ampla variedade de serviços e recursos para desenvolvimento, hospedagem, análise de dados, inteligência artificial, segurança e gerenciamento de infraestrutura.

A plataforma também oferece opções de personalização no portal, permitindo ao usuário configurar tema, idioma de preferência, formato regional, organização do layout e comportamento do menu.

Entre as diversas categorias de serviços disponíveis no Azure, destacam-se:

- IA + Machine Learning
- Análise
- Computação
- Contêineres
- Bancos de dados
- Redes
- Segurança
- Armazenamento
- DevOps
- Web e Mobile

## Sobre computação em nuvem

Computação em nuvem é a oferta de recursos de tecnologia pela internet, como servidores, armazenamento, bancos de dados, redes e software. Em vez de depender apenas de infraestrutura local, é possível usar esses recursos sob demanda, com escalabilidade, agilidade e melhor aproveitamento de custos.

Nesse contexto, o Azure atua como uma das principais plataformas do mercado, oferecendo serviços que atendem desde aplicações simples até arquiteturas complexas e corporativas.
